English

by Laurel Newton


Description

This flipchart helps teach phonics using the Soundwave chart in a Year 1 classroom. Features of these lessons include

  • The Soundwave chart
  • A link to the soundwave YouTube chant video
  • Say the word
  • Use the word in a sentence
  • Phoneme box
  • Spell the word using letter names
  • Dictionary activity
  • Where to find the spelling choice on the chart
  • Hands on small independent group activities
